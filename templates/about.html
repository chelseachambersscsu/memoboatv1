About MemoBoat

Built for people who
hate rewriting notes.

MemoBoat is a full-stack web application that uses the OpenAI API to transform raw, unstructured meeting notes into clean, professional memos. Memos include a summary, action items, and key decisions.

How It Works

  • Paste your raw meeting notes into the input panel on the home page
  • Select a meeting type to categorize your memo
  • Hit Generate Memo. The OpenAI API processes your notes and returns structured output
  • Review your summary, action items, and key decisions in the preview panel
  • Every memo is saved automatically to the database and accessible from the Saved Memos page
  • Download any memo as a text file

Tech Stack

  • Python & Flask for the backend and routing
  • SQLite database with three related tables
  • Jinja2 templates for dynamic HTML rendering
  • OpenAI API for AI-powered memo generation
  • Bootstrap 5 for responsive layout
  • JavaScript for event handling
